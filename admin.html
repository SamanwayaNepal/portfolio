Backup & restore

Because the website is hosted on GitHub Pages (a static host), your edits, events, and notices are stored in this browser only. To publish them for all visitors or move them to another device:

  1. Export the content file below.
  2. On another device, sign in to this dashboard and import the file — or share it with the site maintainer to commit it to the repository.
